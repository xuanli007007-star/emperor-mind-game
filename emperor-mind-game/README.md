# 帝王识人术 · 互动修炼场（两章 · 无AI版）

## 本地运行
```bash
npm i
npm run dev
```
打开 Vite 提供的本地链接即可。

## 一键在线试玩（推荐）
- 访问 https://stackblitz.com
- 选择 **Vite + React** 模板，新建项目
- 上传本仓库所有文件（或把 `src` 与 `index.html` 内容复制进去）
- 保存后自动生成在线试玩链接（可手机打开）

## Vercel 正式部署
1. 将本项目推送到 GitHub 仓库
2. 登录 https://vercel.com → New Project → Import from GitHub
3. Framework 选择 React/Next.js（Vite 默认即可）
4. 一键 Deploy，拿到正式域名
